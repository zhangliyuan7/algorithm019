##### 位运算
    ```
        >>  0110 -> 0011
        <<  0110 -> 1100
        n&n-1 清除最后一位的1 
        n&(-n)   ==  x & (~x + 1)
             ① 这个结果只有一位值是1， 其他位均是0 
             ② 这个值的末位0的个数与原值保持一致
            
        ^异或  相同为0 不同为1 
        x^0 = x
        x^1s = ~x  1s=～0  全零
        x^(~x) = 1s
        x^x = 0
        c=a^b => a^c =b b^c=a
        a^b^c=a^(b^c)=(a^b)^c
        x&~x = 0   
        
        x最右边的n位清零 x&(~0<<n)
        x的第n个位置 (x>>n)&1
        x的第n位的幂值 x&&(1<<n)
        将第n位置为1  x|(1<<n)
        第n位置换为0  x&(~(1<<n))
        x的最高位至第n位清零  x&((1<<n)-1)
        判断奇偶
            x&1==1 奇数
            x&1==0 偶数
        整除 
            x/2 == x>>1
        清零最低位的1
            x&(x-1)
        得到最低位的1
            x&-x
        
    ```
##### 2的幂次 231
    ```
        二进制有且仅有一个1 即为2的幂次
        二进制位消末位1 n&(n-1) 的次数即个数
        if n!=0 {
            count++
            if count>1{
                return false 
            }
            n=n&(n-1)
        }
        简化：
            if n<0{return false}
            return n&(n-1)==0
    ```
